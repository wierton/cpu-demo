## 环境
1. npm 6.14.13
2. node v14.17.3
3. 脚手架 create-react-app

## 安装依赖
```
npm install
```

## 运行
```npm start```

## 文件说明
### 代码
**右击**CP0模块，展示代码。  
CP0模块代码为/src/resource/core/cp0.scala文件。其他模块以此类推。

### 信号
信号为/src/resource/sample-signals.txt

### VGA
VGA为/src/resource/VGA.txt

### log
log为/src/resource/sample-serial.txt  

### 配置滑动条速度
在/src/config/config.js下设置
